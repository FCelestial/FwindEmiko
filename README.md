## 👋 你好，朋友。

我是 **狐风轩汐**（F.windEmiko）—— **锐界幻境（MiragEdge）** 的创始人兼开发者。  
热爱 Minecraft 插件开发，用代码搭建一个 Java 版与基岩版玩家能同台游戏的互通世界。

> 🌌 创造一个让每个玩家都能找到归属感的幻境。
