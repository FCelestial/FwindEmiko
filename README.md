## 🌊 关于我

我是 **狐风轩汐**（F.windEmiko）。

锐界幻境（MiragEdge）创始人，Minecraft 插件开发者，兼基础设施运维与网络工程师。  
喜欢用代码构建可靠且富有创造力的数字空间——无论是游戏世界的互通幻境，还是底层的网络架构。

> *Crafting code with a fox's wit, the wind's flow, and the tide's calm.*
